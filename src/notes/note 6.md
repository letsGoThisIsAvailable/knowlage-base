layer 1 - Physical -> data cables, CAT6
layer 2 - Data -> switching, mac adresses
layer 3 - Networking - IP adresses, routing
layer 4 - Transport - TCP/UDP
layer 5  - Session - session management
layer 6 - Presentation - media
layer 7 - Application HTTP, SMPT

when troubleshooting go from layer 1 to 7
