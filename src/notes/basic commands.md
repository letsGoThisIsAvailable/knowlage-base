**sudo_____** - give more for the command (requires password and being in 'sudo-ers' file to use)

**clear** - clear terminal

**man ___** - shows an explains a comand  in better detail

**apt-update** - update anything that needs upgrading

**apt install ____** - install something in terminal

**apt-upgrade** - upgrades packages that need it.

**update db** - update terminal's database

**cd  ____**- change directory to only argument
**cd ..** - go back one step in path
