### *TCP* protocalls

*ftp* - represents file transfers.
*SSH* - log into machine remotely [encrpted]
*telnet* - log into file remotely [not encrypted]
*DNS* - converts something like url to ip adress.
*HTTPS / HTTP* - reprsents a website
*SMB* - file shares (and super mario bros).

### *UDP* protocalls

*DNS* - converts something like url to ip adress.
*DHCP* - assigns an ip adress.
